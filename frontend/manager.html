📋 处理报修流程

  1. 点击卡片进入报修详情页
  2. 在详情页可更新状态（待处理→处理中）
  3. 维修完成后，点击「上传完工照片并完成」
  4. 必须上传至少1张完工照片才能结单
  5. 完成后等待业主确认

📷 上传完工照片

  1. 在报修详情页底部「上传完工照片」区域
  2. 点击上传区域选择照片（至少1张）
  3. 点击「上传完工照片并完成」按钮
  4. 上传后状态自动变为「已完成」

👥 用户审核（管理员）

  1. 点击「用户审核管理」进入管理页面
  2. 查看待审核用户提交的证件照片
  3. 点击「批准」或「拒绝」

🔔 状态说明

  • ● 待处理 — 新提交，等待物业接单
  • ● 处理中 — 物业已开始处理
  • ● 已完成 — 物业已完工，等待业主确认
  • ● 已确认 — 业主已确认，工单关闭
